A collection of AI agents designed to perform various specialized tasks. The agents are built using Python and FastAPI, with a focus on modularity and extensibility.

## Features

- **Multiple Specialized Agents**: Finance, Legal, Retail, Pricing, Healthcare, and Real Estate
- **Standardized API**: All agents expose consistent endpoints
- **Agent Registry**: Centralized discovery and management
- **Web Integration**: Connect to web frontends with ease
- **Extensible Architecture**: Easily add new agent types

## Installation

To install the agents, clone this repository and install the required dependencies:

```bash
git clone https://github.com/rcastrejon91/ai_agents.git
cd ai_agents
pip install -r requirements.txt
```
